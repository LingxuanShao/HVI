* README

1. The file `estimation.R` contains the code used for generating the point estimate in Section 5.1.

2. The file `inference.R` contains the code for computing the confidence interval and band in Section 5.2.

3. The file `dat.RDS` supplies the real data in Section 6.

4. The file `realdata.R` provides the code for producing the point estimate, confidence interval and band in Section 6.

5. The file `realdata-PVI.R` offers the code for calculating the point estimate and confidence interval under PVI setting in Section 6.
